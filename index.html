How to generate this mirror

  1. Install dependencies:
    pip install requests beautifulsoup4 tqdm
  2. Run from the repository root:
    python3 scripts/mirror.py
  3. Serve locally to verify:
    python3 -m http.server 8080
  4. Commit & push all generated files.

See README.md for full documentation and scripts/mirror.py for the mirror implementation.
